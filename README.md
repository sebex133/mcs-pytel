# mcs-pytel

MCS-PYTEL company website

Simple HTML project built on Gulp pack

# WARNING - may cause Gulp errors on Chrome (use MS EDGE)

# Develop the project using one of below command (WYSYWIG):
npm run dev
npx gulp

# Run the simple local server using:
cd ./dist
# then one command from below:
php -S localhost:8000
python3 -m http.server 8000
http-server -p 8000
